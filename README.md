# Лабораторная работа: Архиватор файлов

## Цель

Реализовать программу для упаковки и распаковки файлов.

---

## Описание

Необходимо создать собственный архивный формат (без использования zip/tar библиотек).

Программа должна:
- собирать файлы в один архив
- восстанавливать их обратно
- добавлять файлы в архив/удалять
---

## Команды

### 1. Создание архива

```bash
python archiver.py create <имя архива> <абсолютный путь до папки>
```

Архивирует папку 
создаёт файл-архив
ГАРАНТИРУЕТСЯ, ЧТО В ПАПКЕ НЕ БУДЕТ ПОДПАПОК

---

### 2. Распаковка

```bash
python archiver.py unpack <имя архива> <абсолютный путь до папки>
```

Извлекает файлы в папку
---

### 3. Добавить файл в архив

```bash
python archiver.py add <имя архива> <абсолютный путь до файла>
```

Добавляет файл в архив
---

### 4. Удаление файла из архива

```bash
python archiver.py rm <имя архива> <абсолютный путь до файла>
```

Удаляет файл из арихва

---

## Требования

- Умение работать с бинарными файлами
- Внутреннее устройство архивов
